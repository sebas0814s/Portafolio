# Sebastián | Backend Developer

Portafolio personal desarrollado con HTML, CSS y JavaScript vanilla. Diseño moderno estilo developer con temática dark y acentos en verde.

## Tech Stack

- **HTML5** - Semántico y accesible
- **CSS3** - Custom properties, grid, flexbox, animaciones
- **JavaScript** - Vanilla (sin frameworks)
- **Google Fonts** - JetBrains Mono + Inter

## Estructura

```
Portafolio/
├── index.html    # Main HTML
├── style.css    # Estilos
├── script.js    # Interacciones
├── cv.pdf       # Currículum
└── vercel.json  # Config Vercel
```

## Features

- Diseño responsive (mobile, tablet, desktop)
- Animaciones de scroll reveal
- Typed text animado en el hero
- Código en ventana tipo editor
- Modo oscuro
- Navegaciónfixed con efecto scroll
- Proyectos con enlaces a GitHub
